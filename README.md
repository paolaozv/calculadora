# calculadora
Calcular la suma, resta, multiplicación, división, número mayor, menor o igual de dos números.
###Pseudocódigo Caso Switch 1
1. Inicio 
2. Ingrese opción
3. Evaluar si opción es válida
  * No "Opción no válida"
  * Sí Ingrese dos números
4. Según la opción ingresada operar.
5. Mostrar resultado
6. Fin
